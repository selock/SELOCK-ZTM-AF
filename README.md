# SELOCK Zero Trust Mobile Attestation Framework (ZTM-AF)

O **ZTM-AF** é um framework open-core para construção de imagens Android hardened com validação criptográfica de integridade e attestation soberana offline.

## 📂 Estrutura do Projeto

- **/build**: Pipeline de build determinístico e reprodutível.
- **/attestation**: Motor de geração de provas de integridade (.ztmproof).
- **/compliance**: Automação de relatórios para LGPD, ISO 27001 e NIST.
- **/policies**: Definições de hardening declarativo em YAML.
- **/docs**: Documentação técnica e padrões de engenharia.

## 🚀 Diferenciais Estratégicos

- **Independência de Vendor**: Attestation que não depende de Google Play Integrity ou Apple MDM.
- **Prova Matemática**: Validação de integridade baseada em hashes de hardware e assinaturas criptográficas.
- **Compliance Nativo**: Cada configuração técnica gera automaticamente evidência para auditorias de privacidade e segurança.
